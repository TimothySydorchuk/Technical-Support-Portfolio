# Troubleshooting Playbooks

## Internet Connectivity
1. Check network adapter status
2. Run ipconfig
3. Verify default gateway
4. Ping external IP
5. Check DNS resolution
6. Restart adapter

## Login Issues
1. Verify username
2. Check Caps Lock
3. Confirm account status
4. Reset password
5. Check MFA
6. Attempt login again

## Email Issues
1. Verify internet connection
2. Restart email client
3. Check mailbox quota
4. Test webmail login
5. Reconfigure account

## Printer Offline
1. Verify printer power
2. Check network connection
3. Restart print spooler
4. Reinstall printer driver
5. Test print

## Application Access
1. Check internet connectivity
2. Clear browser cache
3. Try different browser
4. Check application status page
5. Escalate if issue persists
