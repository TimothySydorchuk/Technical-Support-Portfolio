# Case 2 – User Cannot Login

## Issue Summary
User reports being unable to log into the company web application.

## User Report
User receives an "Invalid credentials" error when attempting to log in.

## Initial Analysis
Possible causes:

- Incorrect password
- Caps Lock enabled
- Account locked after multiple attempts
- MFA verification failure
- Application authentication issue

## Troubleshooting Steps
1. Confirmed that the user is entering the correct username.
2. Asked the user to verify that Caps Lock is not enabled.
3. Checked account status in the system.
4. Performed password reset.
5. Asked the user to log in again.

## Resolution
User successfully logged in after password reset.

## Status
Resolved.
