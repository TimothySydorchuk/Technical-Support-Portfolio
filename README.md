# Technical Support & Application Support Portfolio

This repository contains practical troubleshooting scenarios demonstrating structured problem investigation and resolution workflows across both L1 (Technical Support) and L2 (Application Support) levels.

The cases simulate real-world support incidents and illustrate how issues are diagnosed, investigated, and resolved in a ticket-based support environment.

## Skills demonstrated
- Incident troubleshooting
- Structured problem investigation
- SQL data verification and analysis
- API investigation (Postman, request/response analysis)
- Root cause analysis (RCA)
- Ticket documentation and escalation 
- Basic networking diagnostics
- Application and system behavior analysis
