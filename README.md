# technical-support-portfolio

This repository contains practical troubleshooting scenarios demonstrating structured problem investigation and resolution workflows typical for IT Support (L1 / Technical Support) roles.

The cases simulate real-world support incidents and show the approach used to diagnose, troubleshoot and resolve user issues.

## Skills demonstrated
- Incident troubleshooting
- Structured problem investigation
- Ticket documentation
- Escalation decision making
- Basic networking diagnostics
- Application support investigation

## Case Studies
1. Internet Connectivity Issue
2. User Cannot Login
3. Email Not Working
4. Application Access Issue

Each case includes:
- Issue summary
- User report
- Initial analysis
- Troubleshooting steps
- Resolution or escalation
