# Case 3 – Email Not Syncing

## Issue Summary
Email Client Not Syncing New Messages.

## User Report
User reports that new emails are not appearing in the Outlook client. The last email received was several hours ago.

## Initial Analysis
Possible causes:

- Internet connectivity issue
- Email client synchronization problem
- Mailbox quota reached
- Incorrect account configuration
- Server connectivity issue

## Troubleshooting Steps

1. Verified internet connectivity.
2. Restarted the email client.
3. Checked mailbox storage quota.
4. Attempted login via webmail.
5. Verified account configuration in the email client.
6. Triggered manual synchronization.

## Resolution
Email synchronization resumed after restarting the email client and re-establishing the server connection.

## Status
Resolved.
